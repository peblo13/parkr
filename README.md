# Futurystyczna Sala Zabaw dla Dzieci - Wesoły Zakątek

## Opis
Nowoczesna strona firmowa dla sali zabaw dla dzieci w branży rozrywka/kultura. Strona zawiera futurystyczny design, formularz kontaktowy oraz możliwość dodawania komentarzy.

## Funkcje
- Responsywny design
- Animacje CSS
- Formularz kontaktowy (frontend)
- System komentarzy z localStorage
- Galeria zdjęć
- Sekcje: O nas, Usługi, Galeria, Kontakt, Komentarze

## Jak uruchomić
1. Otwórz `index.html` w przeglądarce internetowej.
2. Strona jest statyczna, więc nie wymaga serwera.

## Technologie
- HTML5
- CSS3 (z animacjami)
- JavaScript (ES6)

## Troubleshooting
- Jeśli komentarze nie zapisują się, sprawdź czy przeglądarka obsługuje localStorage.
- Dla pełnej funkcjonalności formularza kontaktowego, dodaj backend (np. PHP lub Node.js).
